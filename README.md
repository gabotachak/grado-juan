# Grado de Juan Diego 🎓

Invitación web para la celebración del grado de Médico de Juan Diego Gallego Tachack.

Construido con [Astro](https://astro.build) y desplegado automáticamente a GitHub Pages mediante GitHub Actions en cada push a `main`.

## Desarrollo local

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
npm run preview
```

## Despliegue

El sitio se publica automáticamente en `https://gabotachak.github.io/grado-juan/` vía el workflow `.github/workflows/deploy.yml`. En GitHub, la fuente de Pages debe estar configurada como **GitHub Actions** (Settings → Pages → Source).
